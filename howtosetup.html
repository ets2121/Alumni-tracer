Step-by-Step Guide

How to Set Up the Alumni System

Complete implementation guide for the Laravel project

1. Prerequisites (PHP)

PHP is required to run the Laravel backend system. Before proceeding, ensure you have PHP installed on your machine.

Run this command in your terminal to check your current PHP version:
php -v

If a version appears (e.g., PHP 8.2), you're ready. If not, download and install PHP.

Official PHP Downloads
Windows Users: PHP is most commonly installed via XAMPP, which also provides MySQL and Apache in one package.
Download XAMPP

2. Composer Installation

Composer is the dependency manager for PHP. It is essential for installing Laravel and all required third-party packages.

Download Composer
Verify the installation by running:
composer -V

3. Enable PHP Extensions

The Alumni System requires the GD (Graphics) and Zip extensions to handle images and file archives.

Find your php.ini file. In XAMPP, it is usually at:
C:\xampp\php\php.ini

You must find these lines and remove the semicolon ; at the beginning to enable them:

- ;extension=gd
+ extension=gd
- ;extension=zip
+ extension=zip
  • 1
    Open php.ini in a text editor (Notepad, VS Code).
  • 2
    Search for gd and zip.
  • 3
    Remove the leading semicolon ;.
  • 4
    Save the file and Restart Apache in the XAMPP Control Panel.

4. Database Setup

Create an empty database to hold the system data before running the automated setup.

Database Name: alumni_system
  • 1
    Open http://localhost/phpmyadmin
  • 2
    Click on the "New" button in the left sidebar.
  • 3
    Enter alumni_system in the database name field.
  • 4
    Click Create.

5. Project Initialization

Run these commands one by one in your terminal to prepare the application environment and data.

Step 1: Install PHP dependencies
composer install
Step 2: Generate application security key
php artisan key:generate
Step 3: Run migrations and populate demo data
php artisan migrate:fresh --seed
Please wait for each command to finish before starting the next one.

6. Run the System

Start the local development server to access the application.

php artisan serve

Once running, open your browser and visit:

http://127.0.0.1:8000
